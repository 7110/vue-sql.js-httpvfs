# vue-sql.js-httpvfs

sample of using [sql.js-httpvfs](https://github.com/phiresky/sql.js-httpvfs) with Vue.js

## Demo

[gh-pages](https://7110.github.io/vue-sql.js-httpvfs/)

## Getting Started

Install

```shell
yarn install
```

Prepare sqlite3

- make `/db` directory in `/public`
- put the **.sqlite3** file into `/public/db`
